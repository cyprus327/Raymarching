# Raymarching

My foray into Ray Marching, mainly inspired by the game Claybook and random things on Shadertoy.

## Features:

- Soft shadows
- Any sdf
- Any operation
- Per-object color
- Interaction with the scene during runtime (placing objects, moving objects)
